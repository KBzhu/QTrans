# task_P7.4 执行文档 - 文件列表组件

## 任务概述

创建 FileList.vue 组件，用于展示申请单的文件列表，支持上传、下载、查看三种模式。

## 子任务清单

### 1. 创建 FileList.vue 组件
- [√] 定义 Props（applicationId, mode, files）
- [√] 实现表格展示（文件名、大小、上传时间、状态、操作）
- [√] 实现不同模式的操作列（upload/download/view）
- [√] 实现文件总大小统计
- [√] 实现空状态展示

### 2. 创建样式文件
- [√] 创建 `file-list.scss`
- [√] 实现表格样式
- [√] 实现操作按钮样式

## 技术要点

- 使用 `a-table` 组件展示文件列表
- 根据 mode 动态显示不同操作按钮
- 使用 `formatFileSize` 格式化文件大小
- 空状态引导用户上传

## 执行时间

预计：2小时

   - 根据模式动态显示操作按钮
   - 文件总大小统计展示
