## 差异清单

### 1) 页面定位差异
- **高保真**：3971_812 展示的是首页 Dashboard 视图，传输类型卡片作为 Dashboard 的一部分（Tab 切换 + 卡片区域 + 公告 + 帮助文档）。
- **文档**：P6.1 要求独立的「选择传输类型」页面，7 种传输类型卡片网格布局。
- **处理**：创建独立的 `SelectTypeView.vue` 页面，复用 Figma 中的 Tab + 卡片视觉风格，但作为独立路由 `/application/select-type`。

### 2) 传输类型数量差异
- **高保真**：当前 Tab（例行申请）下仅展示 2 张卡片（例行申请、例行通道），其他 Tab 内容未展开。
- **文档**：要求 7 种传输类型（绿→绿、绿→黄、绿→红、黄→黄、黄→红、红→红、跨国传输）。
- **处理**：按文档实现全部 7 种传输类型，按 Tab 分组展示（绿区传出 3 种、黄区传出 2 种、红区传出 1 种、外网传入含跨国 1 种），例行申请 Tab 保留 Figma 中的 2 张大卡片。

### 3) 卡片布局差异
- **高保真**：例行申请 Tab 下为 2 列大卡片（818px × 260px），居中图标对 + 标题按钮 + 描述。
- **文档**：要求 3 列网格布局，每卡片含图标、名称、审批层级标签、简短说明。
- **处理**：普通传输类型按文档 3 列网格；例行申请 Tab 按 Figma 2 列大卡片布局。两种布局共存。

### 4) 审批层级标签差异
- **高保真**：卡片无审批层级标签，仅有标题按钮（蓝底白字）和描述文字。
- **文档**：要求每卡片显示审批层级标签（免审/一级/二级/三级），带颜色区分。
- **处理**：按文档补齐审批层级标签，使用彩色圆角标签（绿色免审、黄色一级、红色二级、紫色三级）。

### 5) Tab 栏内容差异
- **高保真**：5 个 Tab（绿区传出、黄区传出、红区传出、外网传入、例行申请），「例行申请」为当前选中态（蓝底白字）。
- **文档**：未明确 Tab 分组，仅要求 7 种类型卡片网格。
- **处理**：按 Figma 实现 Tab 分组切换，将 7 种类型按源安全域归类到对应 Tab。

### 6) 安全域图标差异
- **高保真**：图标使用渐变色块（绿色 `#00bba7→#00c950`、蓝绿色 `#2b7fff→#00bba7`），箭头使用浅绿背景 `#e7f5ef`。
- **文档**：要求「安全域色块组合」，未指定具体颜色。
- **处理**：按 Figma 渐变色实现，扩展黄区（`#f59e0b→#fbbf24`）、红区（`#ef4444→#f97316`）、跨国（`#8b5cf6→#6366f1`）渐变。

### 7) 底部公告/帮助区域
- **高保真**：页面下方有「重要公告」和「帮助文档」两个面板。
- **文档**：P6.1 仅要求传输类型选择，不含公告/帮助。
- **处理**：P6.1 不包含公告/帮助区域，这些已在 Dashboard（P4）中实现。

## 本次实现结论
- **Tab 分组**：按 Figma 视觉实现 5 个 Tab，传输类型按源安全域归类。
- **卡片样式**：复用 Figma 的玻璃拟态风格（`rgba` 背景 + `box-shadow` + 圆角），补齐 hover 上浮效果。
- **审批标签**：按文档补齐，Figma 中未涉及。
- **例行申请**：按 Figma 实现 2 列大卡片布局，其他 Tab 按文档 3 列网格。
- **路由**：新增 `/application/select-type` 独立路由，`hidden: true` 不显示在侧边栏。
