# 任务：登录页面重设计

## 任务背景
根据设计图重新设计登录页面，提升用户体验和视觉效果。

## 设计要点

### 1. 整体布局
- 居中卡片布局
- 渐变背景（紫色渐变：#667eea → #764ba2）
- 背景装饰圆形元素

### 2. 登录卡片
- 白色卡片，圆角 16px
- 阴影效果：0 20px 60px rgba(0, 0, 0, 0.3)
- 内边距：48px 40px

### 3. 标题
- "欢迎登录"
- 字号：28px，加粗
- 颜色：#1e293b
- 居中显示

### 4. Tab 切换
- **域账号登录** / **本地账号登录**
- 未激活：灰色 (#94a3b8)
- 激活：紫色 (#667eea)，底部渐变下划线
- Tab 之间有分隔线

### 5. 表单字段
- **账号输入框**
  - 高度：48px
  - 背景色：#f8fafc
  - 前缀图标：用户图标
  - Placeholder：请输入账号
  
- **密码输入框**
  - 高度：48px
  - 背景色：#f8fafc
  - 前缀图标：锁图标
  - Placeholder：请输入密码
  
- Focus 效果：
  - 背景变白
  - 紫色边框
  - 紫色外发光（3px）

### 6. 记住账号
- 复选框样式
- 颜色：#64748b

### 7. 登录按钮
- 高度：48px
- 渐变背景：#667eea → #764ba2
- 圆角：8px
- Hover 效果：上移 2px + 紫色阴影
